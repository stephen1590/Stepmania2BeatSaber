# Note Sequences

## Classic DDR Patterns

### The Right-Hand Stairs

| Left | Down | Up  | Right |
| ---- | ---- | --- | ----- |
| L    |      |     |       |
|      | R    |     |       |
|      |      | L   |       |
|      |      |     | R     |

### The Reverse Right-Hand Stairs

| Left | Down | Up  | Right |
| ---- | ---- | --- | ----- |
|      |      |     | R     |
|      |      | L   |       |
|      | R    |     |       |
| L    |      |     |       |
